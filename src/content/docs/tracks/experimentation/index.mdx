This track covers the mathematical and statistical foundations of online experimentation — the discipline behind every A/B test run at scale. It is organized as a curriculum starting from the core estimation layer and branching into the major failure modes and corrections that separate rigorous experimentation from cargo-cult statistics.

## Prerequisites

- Basic statistics (means, variances, hypothesis testing concepts)
- Familiarity with the normal distribution
- The [Bayesian Statistics](/gradient-lectures/tracks/bayesian-statistics/) track is complementary but not required

<Callout type="tip" title="Reading order">

Start with **Power & Sensitivity** to understand how tests are designed, then work through **Core Estimation** to understand the fundamental statistical model. The remaining notes cover specific failure modes — you can read them in any order once you have the first two down.

</Callout>
