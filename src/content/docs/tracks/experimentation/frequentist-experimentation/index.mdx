This track covers the statistical foundations of running controlled experiments (A/B tests) in an industrial setting. Notes are ordered to build on each other sequentially.

## Notes

2. [Power & Sensitivity](./power-and-sensitivity/) — sizing experiments to reliably detect real effects
3. [Multiple Testing](./multiple-testing/) — controlling false positives when evaluating many hypotheses
4. [Repeated Looks and Peeking](./repeated-looks-and-peeking/) — always-valid inference and sequential testing
5. [Normality & Resampling](./normality-and-resampling/) — handling non-normal metrics
6. [Spillover and Network Effects](./spillover-and-network-effects/) — SUTVA violations and cluster randomization
7. [Interaction Effects](./interaction-effects/) — concurrent experiments interfering with each other
8. [Evidence Aggregation](./evidence-aggregation/) — combining p-values across independent experiments
